# RideSharing Server


## instructions

to set up the server locally, run:

``` bash 
 yarn install
 
```
create a postgreSQL database and set it up on your `.env` file 
 
```bash 
 DB_NAME = "database-name"
 DB_PASSWORD = "postgres"
  
```

then run 

```bash
 yarn watch

```

and 

```bash
 yarn dev

```

and you're set!


Your server is running on localhost:4000
